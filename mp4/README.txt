----------------------
Command Line Arguments
----------------------

Compile: make

Run: head -[number of lines to read] [input file path] | ./count [value for k(optional, for the k-th frequency moment, by default k=2)]

-------------------------
Code Output
-------------------------

The program will output the exact number of distinct words in the input stream, and the second frequency moment.
